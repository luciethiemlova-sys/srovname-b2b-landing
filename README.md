# Partneri Srovname.cz

B2B partner landing page for Srovname.cz - connecting insurance comparison platform with marketing agencies and PPC specialists.

## 📋 Project Overview

This project contains the landing page and supporting documentation for the **Srovname.cz B2B Partner Program**. The primary goal is to attract marketing agencies and freelance PPC specialists by showcasing Srovname.cz as a powerful customer acquisition tool for their clients in the insurance sector.

## 🎯 Target Audience

- **Marketing Agencies**: Looking for lead generation solutions for insurance clients
- **Freelance PPC Specialists**: Seeking competitive advantages for client campaigns
- **Insurance Brokers**: Wanting to expand their digital presence

- **Partner Registration**: Google Form integration for easy onboarding
